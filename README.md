# GAirWays
A Spring Boot-based application for managing airways with secure JWT authentication and PostgreSQL database integration.

## Overview
GAirWays is a Java-based REST API application built with Spring Boot 4.0.5, designed to handle airways management with enterprise-grade security features and database migrations.

## Features
- **Spring Boot Web API** - RESTful API endpoints for airways management
- **Security** - JWT token-based authentication
- **Database** - PostgreSQL support with Flyway database migrations
- **Data Persistence** - Spring Data JPA for database operations
- **Validation** - Jakarta validation API for input validation

## Tech Stack
- **Framework**: Spring Boot 4.0.5
- **Language**: Java 21
- **Database**: PostgreSQL
- **Authentication**: Spring Security with JWT tokens
- **ORM**: Spring Data JPA
- **Build Tool**: Maven
- **Build Helper**: Spring Boot Maven Plugin
- **Migrations**: Flyway with PostgreSQL support

## Prerequisites
- Java 21 or higher
- Maven 3.6+
- PostgreSQL database
